# 【1】机器学习中的关键组件 

1）关键组件包括：

- 可以用来学习的数据；
- 如何转换数据的模型；
- 一个目标函数：用来量化模型的有效性；
- 调整模型参数：优化目标函数的算法；

2）学习的定义：模型自主提高完成某些任务的性能；
